### 7. Difference between DevOps and MLOps?

```text
DevOps mainly focuses on application deployment automation,
infrastructure management, and software delivery.

MLOps extends DevOps principles into machine learning systems,
where additional challenges exist like:
- data versioning
- model tracking
- experiment management
- model retraining
- feature drift
- model monitoring

In MLOps, apart from application deployment,
we also manage the ML lifecycle.
```

### 10. What is MLflow?

```text
MLflow is an open-source MLOps platform used for:
- experiment tracking
- model versioning
- artifact management
- model registry

It helps teams compare model runs,
track metrics and parameters,
and manage the ML lifecycle.
```

---

### 11. What is Airflow?

```text
Apache Airflow is a workflow orchestration tool.

It is used to automate and schedule pipelines using DAGs.
In MLOps, Airflow can orchestrate:
- data ingestion
- preprocessing
- model training
- evaluation
- deployment workflows
```

---

### 12. What is DVC?

```text
DVC stands for Data Version Control.

It is used for:
- dataset versioning
- ML pipeline reproducibility
- tracking model artifacts

It works along with Git for managing ML projects.
```
