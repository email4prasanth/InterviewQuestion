# 🔍 Core Differences

## ☁️ Cloud Architect
**Focus:** Design (big picture)

- Designs cloud infrastructure (AWS/Azure/GCP)
- Chooses services and architecture patterns
- Plans scalability, cost, security, high availability, and compliance

👉 Think:  
“What should we build and how should it be structured?”

---

## ⚙️ DevOps Engineer
**Focus:** Build & automate

- Sets up CI/CD pipelines
- Automates deployments
- Manages infrastructure using IaC (Terraform, etc.)
- Works between development and operations teams

👉 Think:  
“How do we build, deploy, and automate this efficiently?”

---

## 🔧 SRE (Site Reliability Engineer)
**Focus:** Reliability & production stability

- Monitoring, alerting, and logging
- Defines SLAs / SLOs
- Handles incidents
- Performs performance tuning

👉 Think:  
“Is the system reliable, fast, and always available?”

---

## 🏗️ Platform Engineer
**Focus:** Internal developer platform

- Builds reusable tools and platforms for developers
- Works on Kubernetes platforms and internal tooling
- Enables self-service infrastructure

👉 Think:  
“How can we make developers faster and more independent?”
